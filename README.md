# twitter俳句ジェネレータ
twitterの縦画像比率に対応した俳句画像を生成するプログラムです
## 使用ライブラリ
`Flask/
Pillow/
Bootstrap`
## 依存関係のインストール
`pip install flask`  
`pip install pillow`
